# Cyber Shield Chatbot

## PROG6221 Programming Portfolio of Evidence (POE)

### Overview

Cyber Shield Chatbot is a Windows Forms application developed in C# to educate users about cybersecurity awareness. The chatbot provides interactive guidance on common cyber threats, including phishing, password security, safe browsing, privacy, and two-factor authentication.

The project demonstrates object-oriented programming principles, event-driven programming, graphical user interface development, and cybersecurity awareness in a user-friendly application.

## How the Application Works

1. Launch the application.
2. Enter your name.
3. Start the chatbot.
4. Ask cybersecurity-related questions.
5. Receive educational responses.
6. Navigate between the available tabs.
7. Complete quizzes and manage cybersecurity tasks.
